Privacy Policy

This application is a personal, server-side bot designed to automatically upload the owner's videos from a private Google Drive to the owner's TikTok account.

Data Usage:

  • The application accesses videos from a specified Google Drive folder.
  • The application uses the TikTok API to upload these videos to the owner's account.
  • No user data is stored, shared, or collected from any third party. The application only handles the owner's data for the sole purpose of automated posting.
