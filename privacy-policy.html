Privacy Policy

Last updated: June 17, 2025

1. Introduction

This application ("the App") is a personal, server-side bot designed for the sole purpose of automating video uploads for its owner. This policy outlines how data is handled within this limited scope.

2. Data We Access

To perform its function, the App requires access to the following data, granted solely by its owner:

  • TikTok Account Access: The App uses an OAuth 2.0 token to gain permission to upload videos to the owner's TikTok account. It only requests the `video.upload` scope.
  • Google Drive Access: The App uses an OAuth 2.0 token to access and read video files from a specific, pre-authorized folder in the owner's Google Drive.

3. How We Use Data

The accessed data is used exclusively for the App's single function:

  1. The App runs on a schedule (via AWS Lambda).
  2. It authenticates with the Google Drive API to download a single video file.
  3. It then authenticates with the TikTok API to upload that video file.

4. Data Storage and Security

Authentication tokens (for TikTok and Google Drive) are stored securely as encrypted environment variables in the AWS Lambda environment. Video files are processed in-memory and are not stored on any server after the upload process is complete.

5. Data Sharing

No data is ever collected from users or shared with any third party. The App does not have any users other than its owner/developer. The data flow is strictly between the owner's authorized services.
